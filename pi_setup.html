Setting up the Raspberry Pi

Go

Steps:
sudo apt-get install mercurial
sudo vi /etc/mercurial/hgrc
 (add the lines)
[web]
cacerts = /etc/ssl/certs/ca-certificates.crt
 Get the GO source code.
cd /usr/local
sudo hg clone -u release https://code.google.com/p/go
 (note: this takes a while)
Then update to the tip so that the ARM support is available:
cd go
sudo hg update -C tip
 Build the go runtime and command:
cd go/src
export GOARM=7
sudo ./all.bash
 (this takes a real long time)
export PATH=$PATH:/usr/local/go/bin
vi hello.go
 (add the lines)
package main
import "fmt"
func main() {
    fmt.Printf("hello, world\n")
}
 Then run it with the go tool:
go run hello.go
 which produces:
hello, world
